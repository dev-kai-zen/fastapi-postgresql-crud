# Run the API (dev)

How to start the app locally with the FastAPI CLI.

---

## 1. Activate the virtual environment

```bash
. .venv/bin/activate
```

## 2. Run FastAPI in development mode

From the directory that contains your `main.py`:

```bash
fastapi dev main.py
```

The CLI watches files and reloads on change.

**Note:** In this repository the app file is [app/main.py](../app/main.py). From the **repository root**, use:

```bash
fastapi dev app/main.py
```

---
